> _Stay curious. Keep learning. Keep rising._

EduRise isn’t just another learning platform.

Built for those late-night “I want more” moments.
For the ones who open new tabs searching for a better future.

## ✨ About the Project

EduRise is a modern, responsive landing page for an online education platform.

It blends clean UI, smooth animations, and structured content to create an experience that feels… motivating.

Not overwhelming.
Not complicated.
Just clear, inspiring, and easy to explore.

## 🚀 What’s Inside

- 🎯 Hero section with a strong call-to-action
- 🔍 Course search UI
- 📊 Live platform stats
- 📚 Category exploration
- 🎥 Interactive learning section (video-focused)
- 🧠 Popular courses showcase
- 👩‍🏫 Instructor highlights
- 📰 Blog & insights section
- 💌 Newsletter subscription
- 📱 Fully responsive design

## 🛠️ Tech Stack

- HTML5 – structured and semantic
- CSS3 – custom styling + responsive layout
- JavaScript (Vanilla) – interactivity
- AOS (Animate On Scroll) – smooth animations
- Boxicons – clean icon system

## 💡 Design Philosophy

This project focuses on:

- clarity over complexity
- emotion over noise
- experience over decoration
  Every section is designed to guide, not distract.
  Every animation has a purpose — to feel alive, not flashy.

## 📸 Preview

> _A platform that feels like progress._

- soft color palette

- smooth micro-interactions
- structured learning flow

## ⚙️ Getting Started

```bash
git clone https://github.com/Tane4ka170/edu-hub-web.git
cd edu-hub-web
```

Open `index.html` in a browser — and that’s it.

No build tools.
No setup headaches.
Just pure frontend.
